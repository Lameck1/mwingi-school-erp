# IPC Domain Ownership

| IPC Namespace | Owner |
| --- | --- |
| `academic`, `report-card`, `reportcard`, `merit`, `promotion`, `jss`, `cbc`, `stream`, `subject`, `term` | Academic Team |
| `reports`, `report`, `scheduler` | Reporting Team |
| `auth`, `user`, `audit` | Security Team |
| `settings`, `system`, `check`, `backup` | Platform Team |
| `data` | Data Platform Team |
| `finance`, `transactions`, `period`, `exemption` | Finance Team |
| `staff`, `payroll` | HR + Payroll Team |
| `student` | Student Records Team |
| `inventory`, `operations`, `hire` | Operations Team |
| `message`, `notifications` | Communications Team |
| `approval` | Workflow Team |

The machine-readable source of truth is [`ipc-manifest.json`](../ipc-manifest.json), generated by:

- `npm run ipc:manifest:generate`
- `npm run ipc:manifest:check`
